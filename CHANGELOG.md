# Changelog

## [2.0.0] - 2026-02-12

### Added
- 14-Agent Company architecture with 4 divisions
- 4 Specialist Desks (Crypto, Weather, Sports, General)
- Claude Sonnet Judge for top candidates
- Parallel analysis pipeline via futures::join_all
- Paper trading with mode-based exits (Scalp/Swing/Conviction)
- Community knowledge sharing via proxy server
- Telegram commands (/status, /stop, /trades, /open, /help)
- 12-hour periodic email reports with category/mode/model breakdowns
- STOP file graceful shutdown
- Docker support
- Web dashboard at localhost:3000
